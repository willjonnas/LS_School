Critério de desenvolvimento:

Ao usarmos a figura de uma Business School, aplicamos a cor azul escuro ligada à gestão e administração, para tornar todo o ambiente bastante identificativo e sóbrio.

Aplicamos uma navegabilidade sofisticada e intuitiva. 

O slider no main screen pretende dar enfase à apresentação e aos valores da escola, mantendo-os sempre presentes.

O utilizador dispõe do menu lateral para aceder a todas as suas funcionalidades que pretenda.   


Recursos utilizados: 

  • Menu lateral;
  • Slider;
  • Base de dados ROOM;
  • API Google Maps;
  • API Weather;
  • Asset html imbutido.


Nota individual: 

Considerando os factores de criatividade e UX, cada um dos developers pretende um nota que seja:

  • nunca menos de 15;
  • mas também não mais que 20.
